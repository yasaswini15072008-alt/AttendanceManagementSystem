# Attendance Management System

## Project Overview
The Attendance Management System is a Python-based application used to manage student attendance records. It allows users to add students, mark attendance, and view attendance reports.

## Features
- Add Student
- Mark Attendance
- View Attendance Report
- Store Attendance Information

## Technologies Used
- Python
- Dictionaries
- Functions
- Loops
- Conditional Statements

## How to Run
1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the command:

python main.py
